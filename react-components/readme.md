# Launch Rules

### 1. Clone the repository

<br>

```bash
$ git clone <link>
```

<br>

### 2. Install all the required dependencies

<br>

```bash
yarn
```

<br>

### 3. Run linting using special command in package.json file

<br>

```bash
yarn lint
```

<br>

### 4. Launch the app in browser

<br>

```bash
yarn dev
```
